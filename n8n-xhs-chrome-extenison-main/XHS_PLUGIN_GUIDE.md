# 小红书作者数据获取插件使用指南

## 功能概述

这是一个Chrome浏览器插件，专门用于获取小红书作者的详细数据，包括用户信息、粉丝统计和笔记内容。

## 主要功能

### 🔍 智能检测
- 自动检测当前页面是否为小红书作者页面
- 只在正确的页面启用"获取作者数据"按钮
- 支持URL格式：`https://www.xiaohongshu.com/user/profile/*`

### 📊 数据获取
获取的数据包括：
- **基本信息**：用户名、用户ID
- **统计数据**：关注数、粉丝数、获赞数
- **笔记列表**：所有笔记标题
- **详细内容**：前10篇笔记的标题、笔记链接、点赞数

### 💾 数据管理
- 数据安全存储在浏览器本地
- 支持多个作者数据同时保存
- 可以查看JSON格式的完整数据
- 支持单个删除和批量清空

## 使用步骤

### 1. 安装插件
1. 下载并构建项目：`npm run build`
2. 在Chrome浏览器中打开：`chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `dist` 文件夹

### 2. 使用插件

#### 步骤1：访问小红书作者页面
- 打开任意小红书作者的个人主页
- 确保URL格式为：`https://www.xiaohongshu.com/user/profile/用户ID`

#### 步骤2：打开插件
- 点击浏览器工具栏中的插件图标
- 如果在正确页面，会显示"检测到小红书作者页面"
- 如果不在正确页面，按钮会置灰并提示"请在小红书作者页面使用"

#### 步骤3：获取数据
- 点击"获取作者数据"按钮
- 插件会自动：
  1. 获取作者基本信息
  2. 收集所有笔记标题
  3. 从笔记卡片获取前10篇笔记的基本信息（标题、链接、点赞数）
  4. 保存数据到本地存储

#### 步骤4：查看数据
- 数据获取完成后会自动打开Side Panel
- 在Side Panel中可以：
  - 查看作者统计信息
  - 展开查看详细数据
  - 查看JSON格式的完整数据
  - 删除不需要的数据

## 注意事项

### ⚠️ 使用限制
- 仅支持小红书作者个人主页
- 需要页面完全加载后再使用
- 获取详细笔记信息需要时间，请耐心等待

### 🔔 状态提示
- 获取过程中会显示"正在获取数据..."
- 成功后会弹出系统通知
- 失败时会显示错误信息

## 故障排除

### 常见问题

**Q: 点击按钮没有反应？**
A: 确保页面完全加载，并且URL格式正确。

**Q: 数据获取失败？**
A: 可能的原因：
- 网络连接问题
- 页面元素未加载完成
- 小红书页面结构发生变化

**Q: 只获取到部分笔记？**
A: 这是正常情况，插件设计为最多获取前10篇笔记的详细信息。

### 技术支持
如遇到问题，请检查：
1. 浏览器控制台是否有错误信息
2. 插件是否正确安装和启用
3. 页面是否为正确的小红书作者页面格式
